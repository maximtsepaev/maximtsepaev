## Проекты

### [Go Final Project](https://github.com/maxi mtsepaev/go_final_project)
Готовое серверное бэкенд-приложение, написанное по итогам курса обучения[cite: 1].
* **Что внутри**:
  * Реализовано REST API для обработки HTTP-запросов[cite: 1].
  * Подключена база данных PostgreSQL, написаны SQL-запросы для сохранения и выборки данных[cite: 1].
  * Настроена JWT-аутентификация для пользователей и middleware для проверки прав[cite: 1].
  * Приложение упаковано в Docker и запускается через Docker-compose[cite: 1].

### [Flashsale](https://github.com/maximtsepaev/flashsale) *(В разработке)*
* **Идея**: Разработка бэкенда для системы мгновенных распродаж, способной обрабатывать одновременные запросы от множества пользователей.
* **Что реализуется**:
  * Разделение логики на сервисы для заказов и склада (микросервисный подход).
  * Использование gRPC для быстрого и типизированного взаимодействия между компонентами.
  * Настройка безопасной работы со складскими остатками в PostgreSQL (SQL constraints), чтобы товар не уходил в минус при конкурентных запросах[cite: 1].
  * Защита от race condition при одновременной покупке одного и того же товара[cite: 1].
